Welcome to **GitHub Actions Zero to Hero**! 🚀

This repository documents my hands-on journey of learning **GitHub Actions** from the fundamentals to advanced CI/CD automation as part of my **90 Days of DevOps** challenge.

## What You'll Learn

* GitHub Actions fundamentals
* Workflow syntax and YAML
* Events and triggers
* Jobs, steps, and runners
* Environment variables and secrets
* Artifacts and caching
* Matrix builds
* Reusable workflows
* Docker integration
* Automated testing
* Continuous Integration (CI)
* Continuous Deployment (CD)
* Best practices for production-ready pipelines

## Repository Structure

Each folder represents a learning milestone with practical examples, exercises, and real-world workflows that can be reused in DevOps projects.

## Goal

The goal of this repository is to build a strong understanding of GitHub Actions by creating practical workflows that automate software development, testing, and deployment processes.
